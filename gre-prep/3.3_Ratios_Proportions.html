Part 1

The Unknown Multiplier

Turning one ratio into three pieces of information

0:30-0:40 | Transition into the core technique.

Unknown Multiplier

One ratio, three facts

A ratio $a:b$ does not give actual values. The actual values are $ax$ and $bx$ for some multiplier $x$.

First Value

$ax$

Second Value

$bx$

Total

$(a + b)x$

0:40-2:00 | The core insight. A ratio 3:5 means actual values are 3x and 5x. This gives us three pieces of information: first value, second value, and total. Use whichever equation the problem gives you information about. Coaching tip: whenever you see a ratio, immediately introduce the constant multiplier x. This avoids fraction-based calculations entirely and makes every ratio problem a simple algebra problem.

Example 1

Using the total

Problem

The ratio of boys to girls is 3:5. If there are 40 students total, how many girls?

Solution

Boys $= 3x$, Girls $= 5x$

$3x + 5x = 40$, so $8x = 40$, $x = 5$

Girls $= 5(5) =$ $25$

2:00-2:50 | Example 1: we know the total is 40. Set up 3x + 5x = 40, solve for x = 5, then girls = 5x = 25.

Example 2

Using the difference

Problem

The ratio of red to blue marbles is 2:7. There are 15 more blue than red. How many total?

Solution

Red $= 2x$, Blue $= 7x$

$7x - 2x = 15$, so $5x = 15$, $x = 3$

Total $= 9x = 9(3) =$ $27$

2:50-3:40 | Example 2: we know the difference. Blue minus red = 15. That gives 5x = 15, x = 3. Total = 9(3) = 27.

Part 2

The Multiples Insight

What MUST be a multiple of what

3:40-3:50 | Transition into the multiples reasoning.

Multiples Insight

Constraining possible values

If the ratio of A to B is $3:5$, then A must be a multiple of 3, B must be a multiple of 5, and $A + B$ must be a multiple of 8.

Example

Ratio of pennies to nickels is 4:5

Total must be a multiple of $9$

Use For

Quantitative Comparison questions

"Cannot be" and "must be true" problems

3:50-5:00 | The multiples insight. If the ratio is 4:5, total must be a multiple of 9. This is powerful for eliminating answer choices and for QC questions where you need to determine if something can be determined.

Part 3

Balancing Multiple Ratios

Making the common term match

5:00-5:10 | Transition into combining two ratios.

Example 3

Combining two ratios

Problem

Dogs to cats is 2:3. Cats to birds is 4:5. What is dogs to birds?

Solution

Common term: cats. LCM(3, 4) $= 12$

Scale: D:C $= 8{:}12$, C:B $= 12{:}15$

Combined: D:C:B $= 8{:}12{:}15$

Dogs:Birds $=$ $8{:}15$

5:10-6:20 | Example 3: combining two ratios. Find the common term (cats), find LCM of its values (3 and 4 gives 12), scale both ratios, then read off dogs to birds = 8:15. Key insight from Leland Plus: when combining ratios with no common term, use separate variables (x and y) for each ratio, then set up an equation from the given relationship to link them.

Example 4

Three-part ratio with a value

Problem

Fish to hamsters is 5:2. Hamsters to lizards is 3:1. If there are 30 fish, how many lizards?

Solution

Balance hamsters: F:H $= 15{:}6$, H:L $= 6{:}2$

Combined: F:H:L $= 15{:}6{:}2$

$15x = 30$, so $x = 2$

Lizards $= 2(2) =$ $4$

6:20-7:30 | Example 4: three-part ratio with a known value. Balance hamsters to get 15:6:2. Then use 15x = 30 to find x = 2. Lizards = 2(2) = 4.

Part 4

Cross-Multiplication

Solving proportions directly

7:30-7:40 | Transition into proportions and cross-multiplication.

Example 5

Basic proportion

Problem

If $\frac{3}{4} = \frac{x}{20}$, what is $x$?

Solution

Cross-multiply: $3 \times 20 = 4 \times x$

$60 = 4x$

Answer: $x = 15$

7:40-8:20 | Example 5: basic cross-multiplication. If a/b = c/d, then ad = bc. Cross-multiply to get 60 = 4x, so x = 15.

Example 6

Applied proportion

Problem

Width to length of a rectangle is 3:7. If the length is 28, what is the width?

Solution

Set up: $\frac{w}{28} = \frac{3}{7}$

Cross-multiply: $7w = 84$

Answer: $w = 12$

8:20-9:10 | Example 6: applied proportion. Set up width/28 = 3/7. Cross-multiply to get 7w = 84, so w = 12.

Changing Ratios

Before and after with the multiplier

Problem

The ratio of boys to girls in a class is 3:5. After 4 more boys join, the ratio becomes 5:5. How many students are in the class now?

Solution

Before: Boys $= 3x$, Girls $= 5x$

After: $3x + 4 = 5x$ (new ratio 5:5 means equal)

$4 = 2x$, so $x = 2$. Original: 6 boys, 10 girls.

Now: $10 + 10 =$ $20$ students

Strategy: Set up the "before" ratio with multiplier $x$. Apply the change, then use the new ratio to solve for $x$.

9:10-10:10 | Changing-ratio problem. Set up the before ratio: boys = 3x, girls = 5x. After 4 boys join, boys = 3x + 4. The new ratio 5:5 means boys = girls, so 3x + 4 = 5x. Solve for x = 2. Now: 10 boys, 10 girls, 20 total.

Key Takeaways

The unknown multiplier method

  • Ratio a:b - Write first $= ax$, second $= bx$, total $= (a+b)x$
  • Use the given info - Total, difference, or a specific value to solve for $x$
  • Multiples insight - Values must be multiples of their ratio parts
  • Balance multiple ratios - Make the common term equal using LCM, then combine
  • Changing ratios - Set up "before" with multiplier $x$, apply the change, use the new ratio to solve
  • Cross-multiply proportions - If $\frac{a}{b} = \frac{c}{d}$, then $ad = bc$
9:10-10:00 | Summary: the unknown multiplier is your primary tool for all ratio problems. Use totals, differences, or specific values. Balance ratios through the common term. Cross-multiply for proportions.

Absolute Value &
the Number Line

0:00-0:30 | Hook: Absolute value shows up in ways you might not expect on the GRE. Most people remember that the absolute value of negative 5 is 5, but the real power is understanding absolute value as distance. When you see it as distance on a number line, tricky problems become straightforward.

Core Concept

Absolute value = distance

$|x|$ is the distance between $x$ and $0$ on the number line. Distance is always positive (or zero).

Same Result

$|5| = 5$ (five units from 0)

$|-5| = 5$ (also five units from 0)

Expressions

$|x - a|$ = distance from $x$ to $a$

$|7 - 3| = |4| = 4$

$|1 - 3| = |-2| = 2$

Part 1

Absolute Value Equations

Two solutions, one concept

2:00-2:10 | Transition: Let's use distance thinking to solve equations.

Example 1

Basic absolute value equation

Problem

$|x - 3| = 5$

The distance from $x$ to $3$ is $5$.

Solution

Going right: $3 + 5 = 8$

Going left: $3 - 5 = -2$

Check: $|8 - 3| = 5$ and $|-2 - 3| = 5$

Answer: $x = 8$ or $x = -2$

Example 2

Equation with a linear expression

Problem

$|2x + 1| = 9$

Solution

Case 1: $2x + 1 = 9$, so $2x = 8$, $x = 4$

Case 2: $2x + 1 = -9$, so $2x = -10$, $x = -5$

Answer: $x = 4$ or $x = -5$

Pattern

For $|\text{expression}| = k$, set up: expression $= k$ and expression $= -k$.

Part 2

Absolute Value Inequalities

Between or outside

4:30-4:40 | Transition: Now let's apply distance thinking to inequalities. This is where it really pays off.

Inequalities

Less than = between, greater than = outside

Less Than

$|x| < k$ means "closer than $k$ to zero"

$-k < x < k$

$|x| < 4$ gives $-4 < x < 4$

Greater Than

$|x| > k$ means "farther than $k$ from zero"

$x < -k$ or $x > k$

$|x| > 4$ gives $x < -4$ or $x > 4$

Watch Out

Less than gives one interval (between). Greater than gives two pieces (outside). Mixing these up is a common GRE mistake.

4:40-5:40 | The two inequality patterns. Less than means x is between negative k and k. Greater than means x is outside, giving two separate pieces. Emphasize: less than = one interval, greater than = two pieces.

Example 3

Inequality with a shifted center

Problem

$|x - 2| \leq 3$

Distance from $x$ to $2$ is at most $3$.

Solution

$x$ is within 3 units of $2$

$2 - 3 \leq x \leq 2 + 3$

Answer: $-1 \leq x \leq 5$

Example 4

Greater-than with a negative center

Problem

$|x + 5| > 2$

Rewrite: $|x - (-5)| > 2$

Solution

Distance from $x$ to $-5$ is more than $2$

$x < -5 - 2$ or $x > -5 + 2$

Answer: $x < -7$ or $x > -3$

Common Traps

Mistakes the GRE expects

Missing Solutions

$|x - 3| = 5$ has TWO solutions. Always solve both cases.

Negative Equals

$|x| = -3$ has NO solution. Distance is never negative.

Flipped Inequality

$<$ gives one interval. $>$ gives two pieces. Do not mix them up.

7:10-8:30 | Three common traps. First: forgetting the second solution in equations. Second: absolute value can never equal a negative number, so no solution. Third: confusing less-than (between) and greater-than (outside) for inequalities.

Key Takeaways

Absolute value & the number line

  • Absolute value = distance from zero on the number line
  • $|x - a|$ = distance from $x$ to $a$
  • Equations: $|\text{expr}| = k$ splits into expr $= k$ and expr $= -k$
  • Less than gives one interval (between $-k$ and $k$)
  • Greater than gives two pieces (outside $-k$ and $k$)
  • No solution if $|\text{something}|$ equals a negative number
8:30-10:00 | Summary: Absolute value is distance. Equations give two solutions. Less-than inequalities give one interval, greater-than give two pieces. No solution if absolute value equals a negative. Think distance, and these problems become much clearer.

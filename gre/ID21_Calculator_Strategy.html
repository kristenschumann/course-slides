The Calculator Question

Should You Use the Calculator?

The GRE gives you an on-screen calculator. But using it on every problem wastes time, and ignoring it when you need it costs accuracy.

Overusing It

Typing in $2 \times 3$ wastes 10 seconds you could spend thinking

Underusing It

Struggling with $847 \div 7$ by hand wastes time and risks errors

Key Idea

Learn WHEN the calculator helps and when it hurts.

The Basics

Know Your Calculator

What It Has

  • Basic operations: +, -, x, /
  • Square root
  • Percent key (%)
  • Memory functions (MC, MR, M+)
  • Parentheses
  • Sign change (+/-)
  • Decimal point

What It Doesn't Have

  • Exponents (no $x^2$ or $x^y$ key)
  • Fractions (displays decimals only)
  • Trigonometry
  • Logarithms

Important

Key Limitation

The calculator shows DECIMALS, not fractions.

If answer choices are fractions, you may need to convert the decimal result back. Know your common fraction-decimal equivalents.

Calculator shows:

0.375

You need to recognize:

$\frac{3}{8}$

Speed Tip

Practice with the official GRE calculator at ets.org before test day. Don't let unfamiliarity slow you down.

When to Use the Calculator

Use the Calculator

5 Times the Calculator Helps

  • Multi-digit multiplication/division: 847 x 23 (use it), 12 x 5 (skip it)
  • Decimal operations: 3.47 + 8.923 + 12.6 (keeps track of place values)
  • Checking your work: Solved by hand? Quick calculator verify
  • Long chains of operations: \$450 x 1.08 x 0.95 x 1.06
  • Square roots of non-perfect squares: $\sqrt{50}$ gives 7.07...

Pro Tip

Memory Function

Use memory for multi-step calculations where you need to store an intermediate result.

M+

Adds the current display value to memory

MR

Recalls the stored value from memory

MC

Clears the memory completely

Speed Tip

Need to use 847 multiple times? Calculate it, press M+ to store, then use MR whenever you need it again.

Part 3

When to Skip the Calculator

5:00 - 5:05 | Section divider. Now the flip side: six situations where mental math is faster.

Skip the Calculator

6 Times Mental Math Wins

  • Simple operations: $8 \times 7 = 56$, $100 - 37 = 63$
  • Operations with 10, 100, 1000: $45 \times 100 = 4{,}500$ (just shift the decimal)
  • Fraction comparisons: Is $\frac{3}{8} > \frac{1}{3}$? Cross multiply: 9 vs 8
  • Perfect squares: $\sqrt{81} = 9$, $\sqrt{144} = 12$ (instant recall)
  • Estimation problems: When answers are far apart, estimate instead
  • Algebra setup: The calculator can't SET UP your equation
5:05 - 6:00 | Walk through each of the six situations. Emphasize that typing simple math into the calculator is slower than doing it in your head. The calculator can't think for you.

Warning

The Time Trap

Every calculator use takes 5-10 seconds just for typing, clicking, and reading the result.

For simple operations, that's slower than mental math. Over a full section, unnecessary calculator use can cost you 2-3 minutes.

Slow Way

Click calculator, type 8, press x, type 7, press = ... 10 seconds

Fast Way

Framework

The 5-Second Rule

Ask yourself: "Can I do this in my head in under 5 seconds?"

YES: Skip the Calculator

Simple operations you can solve instantly.

$8 \times 7$, $100 - 37$, $\sqrt{49}$, $45 \times 100$

NO: Use the Calculator

Complex arithmetic that needs precision.

$847 \times 23$, $3.47 + 8.92 + 12.6$, $\sqrt{50}$

Key Idea

For operations like $24 \times 8$ or $156 \div 12$, if you hesitate more than 2 seconds deciding, just pick one method and commit.

Absolute Value
and the Number Line

0:00 - 0:30 | Hook: Most people remember that |−5| = 5, but the real power of absolute value is understanding it as distance on a number line. When you see it this way, tricky problems become straightforward.

Core Concept

Absolute Value = Distance

$|x|$ = distance from $x$ to $0$

$|5| = 5$ because 5 is five units from 0

$|-5| = 5$ because −5 is also five units from 0

Distance is always positive (or zero). You can't have a negative distance.

$|x - a|$ = distance from $x$ to $a$

Examples

If $x = 7$: $|7 - 3| = |4| = $ 4
7 is 4 units from 3

If $x = 1$: $|1 - 3| = |-2| = $ 2
1 is 2 units from 3

0:30 - 2:00 | Core concept: |x| is distance from x to 0. Both 5 and −5 are five units from 0. Extended: |x − a| is distance from x to a. Examples with x = 7 and x = 1 relative to 3.

Equations

Solving Absolute Value Equations

Example 1

$|x| = 7$

"Distance from $x$ to 0 is 7"

What numbers are 7 units from 0?
$x = 7$ or $x = -7$

Example 2

$|x - 3| = 5$

"Distance from $x$ to 3 is 5"

5 units right of 3: $3 + 5 = 8$
5 units left of 3: $3 - 5 = -2$
$x = 8$ or $x = -2$

Example 3

$|2x + 1| = 9$

The expression $(2x + 1)$ is 9 from 0

$2x + 1 = 9 \Rightarrow x = 4$

$2x + 1 = -9 \Rightarrow x = -5$

$x = 4$ or $x = -5$

For $|\text{expression}| = k$, set up two equations: expression $= k$ and expression $= -k$

2:00 - 4:30 | Equations: |x| = 7 gives x = 7 or x = −7. |x − 3| = 5 means distance from x to 3 is 5, so x = 8 or x = −2. |2x + 1| = 9 splits into two cases. Pattern: always set up two equations.

Inequalities

Less Than = Between

$|x| < 4$

"$x$ is less than 4 units from 0"

What numbers are closer than 4 units to 0?
Everything between −4 and 4.

$-4 < x < 4$

$|x - 2| \leq 3$

"Distance from $x$ to 2 is at most 3"

$x$ is within 3 units of 2:
From $2 - 3$ to $2 + 3$

$-1 \leq x \leq 5$

Pattern: $|x| < k$ means $-k < x < k$ (one interval, between)

4:30 - 6:00 | Less than inequalities: |x| < 4 means x is between −4 and 4. |x − 2| ≤ 3 means x is within 3 of 2, so −1 ≤ x ≤ 5. Pattern: less than = between (one interval).

Inequalities

Greater Than = Outside

$|x| > 4$

"$x$ is more than 4 units from 0"

What numbers are farther than 4 units from 0?
Everything outside the interval.

$x < -4$ or $x > 4$

$|x + 5| > 2$

This is $|x - (-5)| > 2$
"Distance from $x$ to −5 is more than 2"

$x < -5 - 2$ or $x > -5 + 2$

$x < -7$ or $x > -3$

Pattern: $|x| > k$ means $x < -k$ or $x > k$ (two pieces, outside)

6:00 - 7:30 | Greater than inequalities: |x| > 4 means x < −4 or x > 4. |x + 5| > 2 is distance from x to −5 more than 2, giving x < −7 or x > −3. Pattern: greater than = outside (two pieces).

Traps to Avoid

Four Common Mistakes

Trap 1: Forgetting the Second Solution

$|x - 3| = 5$ gives two answers: $x = 8$ and $x = -2$.
Always solve both cases.

Trap 2: Negative Result Issues

$|x| = -3$ has no solution.
Distance is never negative. If you see this, answer is "no solution."

Trap 3: Confusing < and >

Less than (<) = between (one interval)
Greater than (>) = outside (two pieces)
Mixing these gives wrong answers.

Trap 4: Endpoints with ≤ and ≥

Same logic, but include endpoints.
$|x| \leq 4$ includes $-4 \leq x \leq 4$ (closed interval).

7:30 - 9:00 | Four common traps: 1) Forgetting second solution for equations. 2) |x| = negative has no solution. 3) Confusing < (between) and > (outside). 4) Remember endpoints with ≤ and ≥.

Summary

Key Patterns

Core Concept $|x| = $ distance from 0; $|x - a| = $ distance from $a$
Equations $|\text{expr}| = k$ → solve expr $= k$ and expr $= -k$
Less Than $|x| < k$ → between: $-k < x < k$
Greater Than $|x| > k$ → outside: $x < -k$ or $x > k$
No Solution $|\text{something}| = $ negative number

Technique 1

The Multiply-Through Method

✗ Traditional Approach

Find common denominators, convert each fraction, then solve. Slow and error-prone.

Multiple conversion steps with lots of room for arithmetic mistakes.

✓ Multiply Through

Find the common denominator, then multiply the ENTIRE equation by it. Fractions disappear.

One multiplication step, then solve a clean equation with no fractions.

Example 1

Solving a Fraction Equation

Problem

If $\dfrac{1}{3z} - \dfrac{1}{2z} = \dfrac{1}{6}$, what is $z$?

Solution

Common denominator: $6z$

Multiply every term by $6z$:
$6z \cdot \frac{1}{3z} = 2$,   $6z \cdot \frac{1}{2z} = 3$,   $6z \cdot \frac{1}{6} = z$

Solve: $2 - 3 = z$

Answer: $z = -1$

Example 2

Three-Term Fraction Equation

Problem

If $\dfrac{4}{x} - \dfrac{5}{2x} + \dfrac{6}{4x} = 3$, what is $x$?

Solution

Common denominator: $4x$

Multiply every term by $4x$:
$16 - 10 + 6 = 12x$

Simplify: $12 = 12x$

Answer: $x = 1$

Example 3

Budget Word Problem

Problem

Ed and Ned spend $\frac{1}{3}$ of their budget on pizza, $\frac{1}{7}$ on wings, $\frac{1}{5}$ on drinks, and the remaining $\$170$ on beer. What was their total budget?

Solution

Set up: $\frac{1}{3}b + \frac{1}{7}b + \frac{1}{5}b + 170 = b$

Multiply by $(3)(7)(5) = 105$:
$35b + 15b + 21b + 17{,}850 = 105b$

Simplify: $71b + 17{,}850 = 105b$
$17{,}850 = 34b$

Answer: $b = \$525$

Technique 2

When to Flip Fractions

Situation 1

Moving Across =

If $\frac{3}{4}a = b$

Then $a = \frac{4}{3}b$

Situation 2

Dividing Fractions

$\frac{a}{b} \div \frac{c}{d}$

Flip and multiply: $\frac{a}{b} \times \frac{d}{c}$

Situation 3

1 Over a Fraction

$\frac{1}{3/4} = \frac{4}{3}$

$\frac{1}{1/10} = 10$,   $\frac{1}{a/b} = \frac{b}{a}$

Example 4

Complex Fraction Simplification

Problem

Simplify: $\dfrac{\frac{1}{3}}{\frac{2}{3} - \dfrac{(1-2)}{\frac{1}{3}}}$

Solution

Inner fraction: $\dfrac{-1}{1/3} = -1 \times 3 = -3$

Denominator: $\frac{2}{3} - (-3) = \frac{2}{3} + 3 = \frac{11}{3}$

Full fraction: $\frac{1/3}{11/3} = \frac{1}{3} \times \frac{3}{11}$

Answer: $\dfrac{1}{11}$

Example 5

Combining Algebraic Fractions

Problem

What is $\dfrac{1}{x} + \dfrac{1}{y}$ equivalent to?

Solution

Set $\frac{1}{x} + \frac{1}{y} = \text{?}$ and multiply through by $xy$

$(xy)\frac{1}{x} + (xy)\frac{1}{y} = \text{?}(xy)$
$y + x = \text{?}(xy)$

Answer: $\dfrac{x + y}{xy}$

Technique 3

Fraction Behavior Rules

Positive Fractions
  • Bigger numerator: bigger fraction
    $\frac{2}{5} > \frac{1}{5}$
  • Bigger denominator: smaller fraction
    $\frac{1}{5} > \frac{1}{6}$
Negative Fractions (Opposite!)
  • Bigger numerator: smaller fraction
    $-\frac{3}{5} < -\frac{2}{5}$
  • Bigger denominator: bigger fraction
    $-\frac{1}{6} > -\frac{1}{5}$
8:25-9:30 | Number properties overview. Positive fractions: bigger numerator = bigger, bigger denominator = smaller. Negative fractions: OPPOSITE rules apply. Why? Closer to zero = larger for negatives. Sign rules: same signs = positive, different signs = negative.

Example 6

Quantitative Comparison

Problem

$x$, $y$, and $z$ are non-zero integers and $\dfrac{x}{y} > \dfrac{z}{y}$

Quantity A

$x$

vs

Quantity B

$z$

Analysis

Try $y > 0$: $x=3, z=2, y=1$
$\frac{3}{1} > \frac{2}{1}$ ✓   So $x > z$

Try $y < 0$: $x=2, z=3, y=-1$
$\frac{2}{-1} > \frac{3}{-1}$, i.e. $-2 > -3$ ✓   So $z > x$

The trap: when $y$ is negative, the inequality flips!

Answer: D (Cannot be determined)

Example 7

Greatest Value of a Fraction

Problem

For which value of $x$ is $\dfrac{1}{(-x)^3}$ greatest?

(A) $-3$
(B) $-\frac{3}{2}$
(C) $0$
(D) $1$
(E) $\frac{3}{2}$

Analysis

For positive result: $(-x)^3$ must be positive, so $x$ must be negative

Need $(-x)^3$ small and positive for the fraction to be large

$x = -\frac{3}{2}$: $(\frac{3}{2})^3 = \frac{27}{8}$, giving $\frac{1}{27/8} = \frac{8}{27}$

Answer: B

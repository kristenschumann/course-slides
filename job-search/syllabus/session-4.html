Session 4 of 6

Build an AI Project That Gets You Hired

Build a portfolio-ready AI project that demonstrates practical business value to hiring managers. Tonight you build something real, publish it, and learn how to talk about it in interviews.

Deliverable: A working AI project on GitHub + LinkedIn showcase post
90 min live session ~85 min homework

Come Prepared With

  • Your AI project idea (at least a rough concept of the problem you want to solve)
  • An AI tool ready to go (Claude, ChatGPT, or similar)
  • GitHub account logged in and accessible
  • A sample dataset or input for your project (even a rough example works)

Session Overview

Learning Objectives

  • Scope an AI project that demonstrates practical business value
  • Build a working automated workflow using Claude Code or similar AI tools
  • Document and present your project for GitHub portfolio and LinkedIn showcase
  • Craft the "tell me about a time you used AI" interview answer using your project

90-Minute Agenda

5 min

Check-In

Week 3 wins, callback check, momentum building from optimized applications.

10 min

Why AI Projects Win Interviews

The AI experience gap: 90% of companies investing in AI, under 25% of candidates can demonstrate hands-on project work. What hiring managers want in three words: real problem, working solution, clear communication.

15 min

Live Build Demo

Watch the coach build a complete AI workflow from scratch. A meeting notes summarizer, start to finish. See the full process: define the problem, write the prompt, run it, refine once, save for reuse.

35 min

Hands-On: Build Your Project

Choose your project, build in pairs, refine your output, then volunteer demos. Two phases: first run (12 min) and refinement (12 min), plus quick shares at the end.

15 min

Document and Showcase

Three places to showcase your project: GitHub README (5-section template), LinkedIn post (hook + story structure), and interview answer (STAR format). Write all three.

10 min

Wrap-Up and Preview

Celebrate what you built. Homework assignments. Preview Session 5.

A Working AI Workflow

GitHub Repository with Professional README

Your project published on GitHub with a 5-section README: problem, solution, how to use it, results, and what you learned.

Template: LinkedIn post (coming soon)

Interview-Ready Answer

A concrete response for "Tell me about a time you used AI." Your project becomes a STAR story with a measurable result.

Homework

Complete these before Session 5. The accomplishments list is critical prep for interview sessions.

This Week's Assignments

~85 min over the week

Polish and publish your AI project 30 min

Clean up your code, finalize your README, and make sure the GitHub repo is public. A polished project with a clear README makes a stronger impression than a complex project with no documentation.

Publish your LinkedIn post 10 min

Take the draft you wrote in session and publish it. AI project posts get strong engagement right now. Link to your GitHub repo. Use relevant hashtags.

List 5 work accomplishments with measurable results 20 min

Write down five things you've achieved at work that had a measurable impact. Revenue generated, time saved, team grown, process improved. These become the raw material for STAR-format behavioral answers in Session 5.

Draft your "tell me about yourself" answer 25 min

Write a first draft using whatever structure feels natural. Don't worry about perfecting it. You'll rewrite it with a proven formula in Session 5. Just get something on paper.

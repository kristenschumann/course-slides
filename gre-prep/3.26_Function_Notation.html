Function Notation &
Plug-In Strategies

0:00-0:30 | Hook: If f(x) = x^2 - 3x, what is f(5)? What is f(a+1)? Function notation is just substitution with a fancy name. The GRE uses functions to test whether you can follow instructions precisely. This video makes function notation intuitive.

Core Concept

Functions are substitution machines

The Rule

$f(x) = x^2 - 3x$ means: take the input, square it, then subtract 3 times the input.

Number Input

$f(5)$: replace every $x$ with 5

$(5)^2 - 3(5) = 25 - 15 = $ $10$

Expression Input

$f(a + 1)$: replace every $x$ with $(a + 1)$

$(a+1)^2 - 3(a+1)$

0:30-2:00 | A function is a substitution machine. f(x) tells you the rule. f(5) means replace x with 5. f(a+1) means replace x with the entire expression (a+1). Always use parentheses around the substitution.

Example 1

Plugging in an expression

Problem

If $g(x) = x^2 - 4$, find $g(x + 2)$.

Solution

Step 1: Replace $x$ with $(x + 2)$

Step 2: $(x+2)^2 - 4 = x^2 + 4x + 4 - 4$

Answer: $x^2 + 4x$

Common Trap

$f(a + b) = (a + b)^2$, NOT $a^2 + b^2$. Always use parentheses when substituting.

2:00-3:00 | Example 1: plugging in an expression. Replace x with (x+2). Expand (x+2)^2 = x^2 + 4x + 4. Subtract 4 to get x^2 + 4x. Common trap: forgetting to square the whole expression.

Part 2

Composite Functions

Plugging one function into another

3:00-3:10 | Transition: now let's plug one function into another. This is called composition.

Composition

Read from inside out

$f(g(x))$ means: first apply $g$, then apply $f$ to the result. Work inside out.

f(g(3))

Let $f(x) = 2x + 1$ and $g(x) = x^2$

Inner: $g(3) = 9$

Outer: $f(9) = 2(9) + 1 = $ $19$

g(f(3))

Same functions, different order

Inner: $f(3) = 7$

Outer: $g(7) = 7^2 = $ $49$

3:10-4:30 | Composition: f(g(x)) means inside out. f(g(3)): first g(3) = 9, then f(9) = 19. Reverse: g(f(3)): first f(3) = 7, then g(7) = 49. Order matters!

Example 2

Composing with variables

Problem

If $f(x) = x + 3$ and $g(x) = 2x$, find $f(g(x))$ and $g(f(x))$.

Solution

$f(g(x)) = f(2x) = 2x + 3$

$g(f(x)) = g(x + 3) = 2(x + 3) = 2x + 6$

$f(g(x)) = 2x + 3$   $g(f(x)) = 2x + 6$

4:30-5:30 | Example 2: composing with variables. f(g(x)) = f(2x) = 2x + 3. g(f(x)) = g(x+3) = 2(x+3) = 2x + 6. Different results, confirming order matters.

Part 3

Finding Inputs from Outputs

Working backwards with functions

5:30-5:40 | Transition: sometimes the GRE gives you the output and asks for the input. This means setting up and solving an equation.

Example 3

Finding the input

Problem

If $g(x) = x^2 - 5x$, for what values of $x$ does $g(x) = 6$?

Solution

Step 1: $x^2 - 5x = 6$

Step 2: $x^2 - 5x - 6 = 0$

Step 3: $(x - 6)(x + 1) = 0$

Answer: $x = 6$ or $x = -1$

5:40-6:40 | Example 3: finding the input. Set g(x) = 6 and solve the quadratic. Factor to get (x-6)(x+1) = 0. Two solutions: x = 6 or x = -1.

Example 4

Relating two inputs

Problem

If $f(x) = 2x + 1$ and $f(a) = f(b) + 4$, express $a$ in terms of $b$.

Solution

Step 1: $f(a) = 2a + 1$, $f(b) = 2b + 1$

Step 2: $2a + 1 = (2b + 1) + 4$

Step 3: $2a = 2b + 4$

Answer: $a = b + 2$

6:40-7:30 | Example 4: relating two inputs. Evaluate f(a) and f(b), set up the equation, and solve for a. Answer: a = b + 2.

Example 5

Plug-in strategy for multiple choice

Problem

If $f(x) = x^2$, which equals $f(2a)$?

(A) $2f(a)$
(B) $f(a^2)$
(C) $4f(a)$
(D) $f(a) + f(a)$
(E) $(f(a))^2$

Solution

Plug in $a = 3$: $f(6) = 36$

(A) $2(9) = 18$  (B) $f(9) = 81$

(C) $4(9) = 36$   (D) $9 + 9 = 18$   (E) $81$

Answer: (C) $4f(a)$

7:30-8:45 | Example 5: plug-in strategy. Plug in a = 3. f(2a) = f(6) = 36. Test each choice: only (C) gives 36. Verify algebraically: f(2a) = (2a)^2 = 4a^2 = 4f(a).

Strategy

When to use plug-in numbers

Variables in Choices

Answer choices contain variables, not specific numbers.

Percent Problems

No specific values given. Use 100 as your starting number.

"Must Be True"

Test multiple values to eliminate wrong choices quickly.

8:45-9:30 | Three scenarios for plug-in: variables in answer choices, percent problems without specific values, and "must be true" questions. Pick simple numbers like 2, 3, or 100 to make the math easy.

Part 4

Custom Operator Problems

"Weird symbol" questions are pure substitution

9:30-9:40 | Transition: the GRE loves to define made-up symbols. These look intimidating but are just function notation in disguise.

Custom Operators

Substitute and simplify

Problem

For all integers $x$ and $y$:

$x \oplus y = 2x + 3y - xy$

What is $3 \oplus (-2)$?

Solution

Step 1: Replace $x$ with 3, $y$ with $-2$

Step 2: $2(3) + 3(-2) - (3)(-2)$

$= 6 - 6 + 6$

Answer: $6$

Strategy: Custom operator problems are pure plug-in. Substitute carefully, watch your signs, and simplify.

9:40-10:40 | Custom operator example. The symbol is just a function definition. Replace x with 3 and y with -2. Be careful with the signs: 3 times -2 = -6, and subtracting -6 gives +6. Answer: 6.

Nested Operators

Work inside out

Problem

Using $x \oplus y = 2x + 3y - xy$:

What is $(2 \oplus 1) \oplus 3$?

Solution

Inner: $2 \oplus 1 = 2(2) + 3(1) - (2)(1) = 5$

Outer: $5 \oplus 3 = 2(5) + 3(3) - (5)(3)$

$= 10 + 9 - 15 = 4$

Answer: $4$

GRE Tip

Nested operators work just like composition. Evaluate the inner operation first, then use that result in the outer operation.

10:40-11:30 | Nested custom operators. Same idea as function composition: work inside out. First evaluate 2 oplus 1 = 5. Then 5 oplus 3 = 4. These are common on the GRE and are free points if you substitute carefully.

Key Takeaways

Function notation and plug-in

  • Evaluation: Replace every $x$ with the input. Always use parentheses.
  • Composition: $f(g(x))$ means inside out. Order matters.
  • Finding inputs: Set $f(x) =$ target value and solve the equation.
  • Parentheses trap: $f(a+b) = (a+b)^2$, not $a^2 + b^2$.
  • Custom operators: "Weird symbol" problems are just substitution. Replace variables carefully.
  • Plug-in strategy: Pick simple numbers to test answer choices when variables appear.
9:30-10:00 | Summary: function evaluation is substitution. Composition works inside out. Find inputs by solving equations. Watch the parentheses trap. Use plug-in with simple numbers for multiple choice.

Multiplication Principle

Sequential choices

If a task has $n$ stages with $a_1, a_2, \ldots, a_n$ choices per stage: Total = $a_1 \times a_2 \times \cdots \times a_n$

Lunch Combo

4 sandwiches, 3 sides, 5 drinks

$4 \times 3 \times 5 = $ $60$ combos

3-Letter Words

26 letters, repeats allowed

$26 \times 26 \times 26 = $ $17{,}576$

0:40-2:00 | The multiplication principle: if a task has multiple stages, multiply the choices at each stage. Walk through the lunch combo example (4 x 3 x 5 = 60) and the 3-letter word example (26^3 = 17,576). GRE tip: draw blank slots on scratch paper and fill in the number of choices for each slot before multiplying. This visual approach prevents errors on complex problems.

Multiplication Principle

Repetition changes the count

When items cannot repeat, available choices shrink at each stage.

With Repeats

3-letter words, repeats OK

$26 \times 26 \times 26 = $ $17{,}576$

No Repeats

3-letter words, no repeats

$26 \times 25 \times 24 = $ $15{,}600$

2:00-3:00 | Compare repetition vs no repetition. With repeats: 26 choices at every position. Without repeats: 26, then 25, then 24, because each used letter is unavailable.

Addition Principle

Mutually exclusive choices

If a task can be done in way A OR way B (not both): Total = ways A + ways B

Keyword Principle Operation
AND, then, followed by Multiplication Multiply
OR, either Addition Add
3:10-4:00 | The addition principle: when choices are mutually exclusive (OR), add them. Key decision: AND means multiply, OR means add. Show the keyword table for quick reference.

Addition Principle

OR means add

Problem

A password is either 4 letters OR 5 digits. How many possible passwords?

Solution

Case 1: 4 letters $= 26^4 = 456{,}976$

Case 2: 5 digits $= 10^5 = 100{,}000$

Answer: $556{,}976$

4:00-4:40 | Example: password is either 4 letters OR 5 digits. These are mutually exclusive cases, so add. Click to reveal solution. Case 1: 26^4. Case 2: 10^5. Total: 556,976.

Part 3

Counting with Restrictions

Handle constraints first, then fill remaining slots

4:40-4:50 | Transition to the restriction strategy: always handle the constrained positions first.

Restrictions

Handle constraints first

Problem

How many 4-digit numbers have no repeated digits and start with an odd digit?

Solution

Restriction first: 1st digit must be odd, so $5$ choices

Remaining: $9 \times 8 \times 7$ (no repeats)

Answer: $5 \times 9 \times 8 \times 7 =$ $2{,}520$

4:50-5:50 | Example: 4-digit numbers, no repeats, starts odd. Handle the restriction first: 5 odd choices for the first digit. Then fill remaining positions with 9, 8, 7 choices. Total: 2,520. Common GRE trap: students forget that "4-digit number" means the first digit cannot be 0. When combined with the odd constraint, this is already handled, but watch for problems where the leading-zero restriction is the only constraint.

Restrictions

Treating a pair as one unit

Problem

5 people line up. A and B must stand next to each other. How many arrangements?

Solution

Treat [AB] as one unit: 4 "items" to arrange $= 4! = 24$

A and B can swap within: $2! = 2$

Answer: $24 \times 2 =$ $48$

5:50-6:50 | Adjacent constraint: treat the pair as one unit. Now arrange 4 items (4! = 24), then multiply by 2 for internal swaps. Total: 48.

Restrictions

The complement strategy

Problem

5 people line up. A and B must NOT be adjacent. How many arrangements?

Solution

Total arrangements: $5! = 120$

Adjacent arrangements: $48$ (from the adjacent constraint)

Answer: $120 - 48 =$ $72$

6:50-7:40 | Complement strategy: when "NOT" is complex, subtract from total. Total 5! = 120 minus adjacent 48 = 72 non-adjacent arrangements. This is one of the most powerful strategies on the GRE. Whenever you see "not," "at least one," or "no two consecutive," try the complement approach first.

Example

The slot method with cases

Problem

How many 5-digit even numbers exist?

Solution

Case 1: Last digit $= 0$. First: 9 choices, middle: $10^3$. Subtotal: $9{,}000$

Case 2: Last digit $= 2,4,6,8$. First: 9, middle: $10^3$. Subtotal: $36{,}000$

Answer: $9{,}000 + 36{,}000 =$ $45{,}000$

7:40-9:00 | Slot method with case analysis. 5-digit even numbers: handle the last digit restriction first. Case 1: last digit is 0 gives 9,000. Case 2: last digit is 2, 4, 6, or 8 gives 36,000. Add cases for 45,000 total.

Key Takeaways

Fundamental counting principles

  • Multiplication principle - Sequential stages (AND): multiply the choices at each stage
  • Addition principle - Mutually exclusive options (OR): add the separate counts
  • Restrictions first - Always fill the constrained position before free positions
  • Adjacent pairs - Treat as one unit, then multiply by internal arrangements
  • Complement strategy - For "NOT" constraints, subtract from total
  • Case analysis - When overlapping restrictions exist, split into non-overlapping cases and add

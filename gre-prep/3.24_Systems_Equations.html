Core Methods

Substitution vs. elimination

Elimination

Add or subtract equations to cancel a variable. This is the default method — reach for it first.

Multiply one or both equations to match coefficients, then add or subtract.

Substitution

Solve one equation for a variable, then plug it into the other.

Use when a variable is already isolated (e.g., $y = 3x$) and elimination is not straightforward.

0:30-1:30 | Two methods: elimination (add/subtract to cancel) and substitution (plug one into the other). Elimination is the default — use it first. Substitution is the fallback when a variable is already isolated or elimination is awkward. Show both methods on this example.

Problem 1

Basic substitution

Problem

Solve for $x$: $y = 3x$ and $x + y = 8$

Solution

Step 1: Substitute: $x + 3x = 8$

Step 2: $4x = 8$

Answer: $x = 2$

1:30-2:15 | Problem 1: basic substitution. y is already isolated, so plug 3x into the second equation. 4x = 8, x = 2.

Problem 2

Adding equations

Problem

Solve: $x + y = 7$ and $x - y = 3$. What is $x$?

Solution

Step 1: Add both equations: $2x = 10$

Step 2: $x = 5$

Answer: $x = 5$

2:15-3:00 | Problem 2: elimination by adding. The y terms cancel: (x+y) + (x-y) = 2x. 2x = 10, x = 5.

Problem 3

Multiply then eliminate

Problem

Solve: $2x + 3y = 13$ and $4x - y = 5$. What is $x$?

Solution

Step 1: Multiply second by 3: $12x - 3y = 15$

Step 2: Add: $14x = 28$

Answer: $x = 2$

3:00-3:50 | Problem 3: multiply then eliminate. Multiply the second equation by 3 to match the y coefficient. Add to get 14x = 28. x = 2.

Problem 4

Ticket word problem

Problem

Adult tickets cost $8 and child tickets cost $5. If 50 tickets are sold for $340, how many adult tickets?

Solution

Step 1: $a + c = 50$ and $8a + 5c = 340$

Step 2: Multiply first by 5: $5a + 5c = 250$

Step 3: Subtract: $3a = 90$

Answer: $30$ adult tickets

3:50-4:50 | Problem 4: ticket word problem. Set up two equations: count and revenue. Multiply the count equation by 5 and subtract to eliminate c. 3a = 90, a = 30.

Problem 5

Finding a combination directly

Problem

If $3a + 4b = 26$ and $2a + 3b = 19$, what is $a + b$?

Solution

Shortcut: Subtract the equations directly

$(3a + 4b) - (2a + 3b) = 26 - 19$

$a + b = 7$

Answer: $a + b = 7$

GRE Tip

When asked for a combination like $a + b$, try adding or subtracting equations before solving individually.

4:50-5:50 | Problem 5: finding a combination directly. Instead of solving for a and b individually, subtract the equations to get a + b = 7 immediately. This is a huge GRE time saver.

Problem 6

No solution (parallel lines)

Problem

For what value of $k$ does this system have no solution?
$2x + ky = 6$
$4x + 6y = 10$

Solution

Step 1: Parallel lines have proportional coefficients

Step 2: $\frac{2}{4} = \frac{k}{6}$, so $\frac{1}{2} = \frac{k}{6}$

Step 3: $k = 3$. Verify: $\frac{6}{10} \neq \frac{1}{2}$, so constants are not proportional.

Answer: $k = 3$

5:50-6:50 | Problem 6: no solution means parallel lines. Coefficients are proportional but constants are not. Set 2/4 = k/6 to find k = 3.

Problem 7

Fraction setup to find a product

Problem

Given: $\frac{x + y}{2} = 5$ and $\frac{x - y}{3} = 2$. What is $xy$?

Solution

Step 1: $x + y = 10$ and $x - y = 6$

Step 2: Add: $2x = 16$, so $x = 8$

Step 3: Substitute: $y = 10 - 8 = 2$

Answer: $xy = 16$

6:50-7:50 | Problem 7: clear fractions first. Multiply to get x+y = 10 and x-y = 6. Add to find x = 8, then y = 2. Product xy = 16.

Problem 8

Using symmetry

Problem

In $ax + by = 8$ and $bx + ay = 10$, if $a + b = 6$, what is $x + y$?

Solution

Step 1: Add both equations

$(a+b)x + (a+b)y = 18$

Step 2: Factor: $(a+b)(x+y) = 18$, so $6(x+y) = 18$

Answer: $x + y = 3$

7:50-8:50 | Problem 8: symmetry trick. Add the equations to get (a+b)(x+y) = 18. Since a+b = 6, divide to get x+y = 3. No need to find individual values.

Key Takeaways

Systems of equations

  • Substitution: Best when one variable is already isolated.
  • Elimination: Add or subtract equations after matching coefficients.
  • Combination shortcut: When asked for $a + b$ or $x - y$, try adding/subtracting equations directly.
  • No solution: Parallel lines have proportional coefficients but different constants.
  • Clear fractions first: Multiply through to eliminate denominators before solving.
  • Look for symmetry: Factor common expressions to avoid solving for individual variables.
8:50-9:30 | Summary: substitution vs. elimination, the combination shortcut, recognizing no-solution systems, clearing fractions, and using symmetry. These techniques save significant time on the GRE.

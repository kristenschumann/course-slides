Methods

The Conversion Chain

Fraction to Decimal

Divide numerator by denominator

$\frac{3}{8} = 3 \div 8 = 0.375$

Decimal to Percent

Multiply by 100 (shift decimal 2 right)

$0.375 \rightarrow 37.5\%$

Percent to Decimal

Divide by 100 (shift decimal 2 left)

$45\% \rightarrow 0.45$

Watch Out

$0.5\% = 0.005$, NOT 0.5. Always count your decimal places carefully when converting percents less than 1%.

Memory Trick 1

The 12.5% Pattern

Each eighth adds exactly 12.5%. Count up from zero:

$0\% \xrightarrow{+12.5} 12.5\% \xrightarrow{+12.5} 25\% \xrightarrow{+12.5} 37.5\% \xrightarrow{+12.5} 50\% \xrightarrow{+12.5} 62.5\% \xrightarrow{+12.5} 75\% \xrightarrow{+12.5} 87.5\% \xrightarrow{+12.5} 100\%$

This means:

$12.5\% = \frac{1}{8}$

$25\% = \frac{2}{8} = \frac{1}{4}$

$37.5\% = \frac{3}{8}$

$50\% = \frac{4}{8} = \frac{1}{2}$

And continuing:

$62.5\% = \frac{5}{8}$

$75\% = \frac{6}{8} = \frac{3}{4}$

$87.5\% = \frac{7}{8}$

$100\% = \frac{8}{8} = 1$

4:35 - 5:15 | The 12.5% pattern is the single most useful memory trick. If you know that each eighth is 12.5%, you can derive any eighth instantly. Walk through the counting pattern.

Memory Tricks

More Patterns to Remember

Nice Fifths

Fifths give clean decimals: 0.2, 0.4, 0.6, 0.8. Just count by 20%.

Repeating Decimal Clue

See .333..., .666..., or repeating 3s/6s? Think thirds or sixths. $0.1666... = \frac{1}{6}$

The Complement Trick

Know one, get the other free:

$\frac{1}{4} = 25\%$, so $\frac{3}{4} = 75\%$

$\frac{1}{8} = 12.5\%$, so $\frac{7}{8} = 87.5\%$

Fraction to Decimal

If you forget, just divide:

$\frac{3}{8} = 3 \div 8 = 0.375$

But memorization is always faster!

5:15 - 5:45 | Four more memory tricks. The complement trick is particularly useful: if you know one fraction in a family, you can derive its complement by subtracting from 100%.

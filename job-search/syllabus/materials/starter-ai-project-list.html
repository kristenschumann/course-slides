Session 4 Resource

Starter AI Project Ideas

15 portfolio-worthy AI projects across industries. Each includes tools needed, time estimate, and quick-start steps.

Beginner 2-4 hours, no coding required

1. Meeting Notes Summarizer

Takes meeting transcripts and generates structured summaries with action items, owners, and deadlines extracted automatically.

Tools: ChatGPT / Claude
Time: 2 hours
Skills Demonstrated: Prompt engineering, AI tool usage
Best For: Anyone in a meetings-heavy role

Quick Start

  1. Record or transcribe your next meeting using Otter.ai, Zoom transcription, or manual notes
  2. Write a prompt that extracts: summary, decisions made, action items with owners, and open questions
  3. Test with 3 different meeting types and refine your prompt for each

2. Email Draft Generator

Paste context about a situation and get polished professional emails written in your voice and tone.

Tools: ChatGPT / Claude
Time: 2 hours
Skills Demonstrated: Prompt engineering, writing
Best For: Sales, BD, customer-facing roles

Quick Start

  1. Collect 5-10 emails you've sent that represent your best writing and voice
  2. Create a system prompt that captures your tone, common phrases, and formatting preferences
  3. Test across scenarios: cold outreach, follow-ups, difficult conversations, and thank-yous

3. Weekly Report Automator

Compiles data from multiple sources into a formatted, consistent weekly report ready to send to stakeholders.

Tools: ChatGPT / Claude + spreadsheets
Time: 3 hours
Skills Demonstrated: Data synthesis, reporting
Best For: Analysts, project managers, ops

Quick Start

  1. Map out every data source you pull from for your current weekly report
  2. Create a prompt that accepts raw data and outputs your exact report format with highlights and trends
  3. Run it for 2 weeks in parallel with your manual process to validate accuracy

4. Job Posting Analyzer

Batch-analyze job postings to find patterns in required skills, experience levels, and salary signals across a target market.

Tools: ChatGPT / Claude
Time: 2 hours
Skills Demonstrated: Research, pattern recognition
Best For: Career changers researching new fields

Quick Start

  1. Collect 10-15 job postings for your target role from LinkedIn, Indeed, or company career pages
  2. Paste them into an AI tool and ask for patterns: most common skills, recurring phrases, experience ranges
  3. Create a summary document showing the "ideal candidate" profile based on real market data

5. Customer Feedback Categorizer

Sort unstructured customer feedback (reviews, surveys, support tickets) into themes with sentiment scores.

Tools: ChatGPT / Claude
Time: 3 hours
Skills Demonstrated: Data analysis, customer insights
Best For: Product, marketing, CX roles

Quick Start

  1. Export 50-100 pieces of customer feedback from any source (reviews, NPS comments, support tickets)
  2. Build a prompt that categorizes each into themes and assigns positive/negative/neutral sentiment
  3. Generate a summary report with top themes, sentiment distribution, and actionable recommendations
Intermediate 4-8 hours, basic tool comfort needed

6. Competitive Intelligence Brief

Auto-research competitors using publicly available information and generate a formatted briefing document with key findings.

Tools: Claude / ChatGPT + web search
Time: 5 hours
Skills Demonstrated: Research, strategy, analysis
Best For: Strategy, marketing, BD

Quick Start

  1. Pick 3-5 competitors and list the information you want to track (pricing, features, positioning, recent news)
  2. Create a structured prompt that organizes findings into a consistent briefing format
  3. Add a comparison matrix and "so what" analysis section that highlights strategic implications

7. Content Calendar Generator

Generate a full month of social media content ideas based on brand voice, industry trends, and content pillars.

Tools: ChatGPT / Claude
Time: 4 hours
Skills Demonstrated: Content strategy, marketing
Best For: Marketing, communications

Quick Start

  1. Define your content pillars (3-5 themes), target audience, and brand voice guidelines
  2. Create a prompt that generates 30 days of posts with hooks, captions, and suggested formats (carousel, video, text)
  3. Add a review layer that checks for variety, balance across pillars, and alignment with upcoming events

8. Interview Prep Bot

A custom AI interviewer that asks role-specific behavioral and technical questions and provides structured feedback on your answers.

Tools: ChatGPT / Claude custom instructions
Time: 4 hours
Skills Demonstrated: HR knowledge, prompt engineering
Best For: HR, recruiting, anyone job searching

Quick Start

  1. Research common interview questions for your target role (behavioral, technical, case-based)
  2. Build a system prompt that acts as a hiring manager, asks one question at a time, and scores answers using STAR format
  3. Add follow-up question logic and a final scorecard that highlights strengths and areas to improve

9. Data Cleaning Pipeline

Automate messy spreadsheet cleanup: deduplication, format standardization, validation rules, and error flagging.

Tools: Claude Code or ChatGPT + Python
Time: 6 hours
Skills Demonstrated: Data management, basic logic
Best For: Analysts, operations, finance

Quick Start

  1. Take a real messy dataset (anonymized if needed) and document every type of error you find
  2. Use AI to generate cleaning rules: standardize dates, remove duplicates, validate email formats, flag outliers
  3. Create a before/after comparison showing data quality improvements with specific metrics

10. Proposal / SOW Generator

Template-based AI that generates client proposals and statements of work from brief intake notes and past examples.

Tools: ChatGPT / Claude
Time: 5 hours
Skills Demonstrated: Business writing, client management
Best For: Consulting, agencies, sales

Quick Start

  1. Collect 3-5 past proposals you've written (or good examples from your industry) as reference templates
  2. Create a prompt that takes client name, problem summary, and scope notes and generates a full proposal draft
  3. Add sections for pricing options, timeline, and terms â€” then test with a real upcoming opportunity
Advanced 8-15 hours, some technical comfort

11. Multi-Source Research Agent

An AI agent that searches multiple sources, cross-references findings, synthesizes insights, and generates a structured research memo.

Tools: Claude Code, web APIs
Time: 10 hours
Skills Demonstrated: Research design, AI orchestration
Best For: Strategy, consulting, policy

Quick Start

  1. Define a research question and identify 3-5 source types (news, academic papers, industry reports, social media)
  2. Build an agent workflow that queries each source, extracts key findings, and flags contradictions
  3. Create an output template: executive summary, findings by source, confidence levels, and recommended next steps

12. Automated Reporting Dashboard

Pull data from APIs, generate visualizations, compile insights, and email weekly update reports to stakeholders automatically.

Tools: Claude Code + Python
Time: 12 hours
Skills Demonstrated: Data viz, automation, basic programming
Best For: Analytics, finance, operations

Quick Start

  1. Pick a report you currently build manually and identify every data source it pulls from
  2. Use Claude Code to write scripts that fetch data, create charts, and format everything into a report template
  3. Add scheduling (cron job or Zapier trigger) and email delivery to make it fully hands-off

13. Document Q&A System

Upload internal documents and ask natural language questions to get cited answers with source references from your own knowledge base.

Tools: Claude Code or OpenAI + embedding
Time: 10 hours
Skills Demonstrated: Information architecture, AI integration
Best For: Legal, compliance, knowledge management

Quick Start

  1. Gather 10-20 internal documents (policies, guides, FAQs) and organize them by topic
  2. Build a system that chunks documents, creates embeddings, and retrieves relevant sections for any question
  3. Add citation tracking so every answer links back to the specific document and section it came from

14. Sales Lead Scorer

Analyze prospect data from CRM exports and score leads based on custom criteria like fit, engagement, and timing signals.

Tools: Claude Code + CRM export
Time: 8 hours
Skills Demonstrated: Sales ops, data analysis
Best For: Sales, BD, revenue operations

Quick Start

  1. Export your lead/prospect data and identify which past leads converted (and which didn't)
  2. Define scoring criteria: company size, industry fit, engagement level, budget signals, timeline indicators
  3. Build a scoring model that ranks current leads and generates a prioritized outreach list with reasoning

15. Process Automation Workflow

End-to-end automation of a multi-step business process, from trigger event through execution to completion notification.

Tools: Claude Code + Zapier / Make
Time: 15 hours
Skills Demonstrated: Process design, systems thinking
Best For: Operations, IT, project management

Quick Start

  1. Map a manual process end-to-end: every step, decision point, handoff, and output
  2. Identify which steps can be automated (data entry, routing, notifications) vs. which need human judgment
  3. Build the automation using Zapier/Make for integrations and Claude Code for the AI decision-making steps

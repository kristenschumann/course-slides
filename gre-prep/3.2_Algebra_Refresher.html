Solving Equations

Three rules for equivalent equations

Add or Subtract

Add or subtract the same value from both sides.

Multiply or Divide

Multiply or divide both sides by the same nonzero value.

Substitute

Replace any expression with an equivalent expression.

2:10-2:40 | The goal is to isolate the variable. These three operations produce equivalent equations. Whatever you do to one side, you must do to the other.

Example 1

One-step isolation

Problem

Solve: $5x - 7 = 28$

Solution

Add 7 to both sides: $5x = 35$

Divide both sides by 5: $x = 7$

Check: $5(7) - 7 = 28$ $\checkmark$

2:40-3:30 | Example 1: simple two-step solve. Click to reveal solution. Step 1: add 7. Step 2: divide by 5. Always check your answer by plugging back in.

Example 2

Multi-step with distribution

Problem

$11x - 4 - 8x = 2(x + 4) - 2x$

Solution

Combine left: $3x - 4 = 2x + 8 - 2x$

Combine right: $3x - 4 = 8$

Add 4, divide by 3: $3x = 12$

Answer: $x = 4$

3:30-4:30 | Example 2: multiple steps. Click to reveal solution. Combine like terms on the left. Distribute and combine on the right. Add 4, divide by 3 to get x = 4.

Example 3

Solving an inequality

Problem

$\frac{4x + 9}{11} < 5$

Solution

Multiply by 11: $4x + 9 < 55$

Subtract 9: $4x < 46$

Answer: $x < 11.5$

Watch Out

When multiplying or dividing by a negative number, flip the inequality sign.

4:30-5:30 | Example 3: inequality. Same steps as an equation, but watch the sign. Multiply by 11, subtract 9, divide by 4. Emphasize: if you multiply or divide by a negative, flip the inequality.

Example 4

Translate and solve

Problem

Five less than three times a number is 22. What is the number?

Solution

"Three times a number": $3x$

"Five less than that": $3x - 5 = 22$

Solve: $3x = 27$

Answer: $x = 9$

6:20-7:10 | Example 4: translate phrase by phrase. "Three times a number" is 3x. "Five less than" means subtract 5. "Is 22" means equals 22. Solve to get x = 9.

Example 5

Two numbers with a constraint

Problem

The sum of two numbers is 45. One is 9 more than the other. What is the larger number?

Solution

Let smaller $= x$, larger $= x + 9$

$x + (x + 9) = 45$, so $2x + 9 = 45$

$2x = 36$, so $x = 18$

Larger $= 18 + 9 =$ $27$

7:10-8:10 | Example 5: two unknowns but we can express both in terms of one variable. Let the smaller be x, larger is x + 9. Their sum is 45. Solve to get 18 and 27.

GRE Algebraic Shortcuts

Plugging in, backsolving, and when to use each

9:00-9:10 | Transition: Now that you have the algebra fundamentals, let's talk about two GRE-specific shortcuts that let you avoid algebra altogether on many problems.

GRE Strategy

Plugging in numbers

When to use: The problem has variables in the answer choices (VIAC). Pick easy numbers, solve, and match.

Problem

If $x$ items cost $d$ dollars, how much do $y$ items cost?

(A) $\frac{xd}{y}$   (B) $\frac{xy}{d}$   (C) $\frac{yd}{x}$

(D) $xyd$   (E) $\frac{d}{xy}$

Solution

Pick: $x = 2$, $d = 10$, $y = 5$

2 items cost $10 $\rightarrow$ each costs $5

5 items cost $5 \times 5 = 25$. Target: $25$

Check (C): $\frac{5 \times 10}{2} = 25$ $\checkmark$

9:10-10:20 | Plugging In is the most powerful GRE shortcut. Whenever you see variables in the answer choices (VIAC), pick easy numbers. Here we pick x=2 items costing d=10 dollars. That means each item costs 5 dollars. So y=5 items cost 25 dollars. Now plug x=2, d=10, y=5 into each answer choice and find which one gives 25. Choice C works. No algebra needed.

GRE Strategy

Backsolving from answer choices

When to use: The answer choices are numbers. Start with choice C (the middle value) and work backward.

Problem

A store sells a shirt at 25% above cost. If the selling price is $40, what is the cost?

(A) $28   (B) $30   (C) $32   (D) $34   (E) $36

Solution

Try (C) $32: 32 \times 1.25 = 40$ $\checkmark$

Done in one step. Answer: $32$

Why start at C? Numeric choices are usually in order. If C is too big, try smaller. If too small, try larger. You narrow to the answer in at most 2 tries.

10:20-11:20 | Backsolving: when answer choices are numbers, work backward. Start with C since choices are usually ordered. If C is too high, go lower; if too low, go higher. Here, C = 32. Mark up 25% gives exactly 40. Done. This avoids setting up the equation entirely.

GRE Strategy

Choosing the right approach

Signal Strategy Example Clue
Variables in answer choices Plug In "In terms of $x$ and $y$"
Numbers in answer choices Backsolve "What is the value of..."
Must-be-true or always-true Plug In multiple values "Which must be true?"
Clean equation, one unknown Algebra $3x + 7 = 22$
11:20-12:00 | Quick decision guide. VIAC = plug in. Numeric answers = backsolve. "Must be true" = plug in multiple values to eliminate. Clean single-variable equation = just solve algebraically. Having all three strategies lets you pick the fastest path on every problem.

Key Takeaways

Algebra toolkit for ratios

  • Combine like terms - Same variable parts, add the coefficients
  • Isolate the variable - Do the same operation to both sides
  • Translate carefully - "Less than" means subtract FROM the quantity
  • Plug In for VIAC - Variables in answer choices? Pick numbers and test
  • Backsolve from C - Numeric answer choices? Start with the middle value
  • Flip for negatives - Multiplying or dividing by a negative flips the inequality
9:00-10:00 | Summary: review the four core algebra skills. These are the building blocks for every ratio and proportion problem coming up next.
